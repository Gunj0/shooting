# ミニシューティングゲーム

## 公開先

- [GitHub Pages](https://gunj0.github.io/shooting/)

## 技術スタック

- Vanilla JavaScript
- HTML/CSS

## 開発ルール

- パッケージ管理は `pnpm` を使用します (`npm` は使用しません)。
- テストは依存関係を最小化する方針とし、原則として Node.js 標準の `node:test` / `node:assert` を使います。

## テスト

- 実行コマンド: `pnpm test`
- 現在は `tests/` 配下でユニットテストを実行します。
